# Compound Knowledge Systems

Knowledge captured in a structured system with typed relations compounds over time. Each new note adds value not just through its own content but through the connections it creates with existing notes. The network effect means that the hundredth note is more valuable than the first because it has ninety-nine potential connection points.

This compounding happens naturally when the system enforces interlinks — atomic notes must connect to existing notes, literature notes must trace to sources, and relation types make the nature of each connection explicit.
